# TaskFlow Kanban

TaskFlow Kanban is a lightweight Kanban-style task manager built with **Angular 21** and **signals**.  
It is designed as a portfolio-ready practice project that demonstrates modern Angular patterns:

- Standalone components
- Feature-based architecture
- Signals for state management
- LocalStorage persistence
- GitHub Pages deployment via GitHub Actions

---

## Live Demo

Deployed on GitHub Pages:

`https://vinaygs911.github.io/Task_Manager/`

---

## Features

- Three-column Kanban board:
  - Todo
  - In Progress
  - Done
- Create tasks with:
  - Title
  - Optional description
  - Initial status
- Move tasks between columns
- Delete tasks
- Task data persisted in **localStorage**, so it survives page refresh
- Clean, responsive layout

---

## Tech Stack

- **Angular 21.1.5**
  - Standalone components
  - Signals for state management
- **TypeScript**
- **LocalStorage** for persistence
- **GitHub Actions** for CI/CD
- **GitHub Pages** for hosting
